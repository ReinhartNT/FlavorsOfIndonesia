Taste of Nusantara: Explore Indonesia's Culinary Delights

Ayam Betutu
Ayam Betutu is made using these primary herbs Galangal, Tumeric, Lemongrass, Kaffir lime leaves, and Bay leaves
Babi Guling
Babi Guling is made using these primary herbs Galangal, Tumeric, Lemongrass, Kaffir lime leaves, Bay leaves, and Coriander seeds
Daging Belacang
Daging Belacang is made using these primary herbs Galangal, Tumeric, Lemongrass, Bay leaves and Kaffir lime leaves
Daging Se'i
Daging Se'i is made using simple herbs which are Lemongrass and Bay leaves
Gudeg
Gudeg is made using these primary herbs Teak leaves, Bay leaves, Galangal, Lemongrass
Jagung Titi
Jagung Titi is primarilly a snack made from pounded corn and uses no specific herbs
Ketoprak
Ketoprak is made of simple herbs which is used in almost every dish which are Garlic and Shallots that is used almost everywhere in the world
Kolak
Kolak is a desert which uses Pandan leaves, Cinnamon, and Lemongrass as it's main ingredients
Rawon
Rawon is made using these primary herbs Black keluak nut, Tumeric, Galangal, Lemongrass, Kaffir lime leaves, and Bay leaves
Sate Lilit
Sate Lilit is made using these primary herbs Lemongrass, Kaffir lime leaves, Galangal, and Tumeric
